React Minesweeper Tutorial Part 2

The Game (React)

If you haven't already worked through React Minesweeper Tutorial Part 1, I suggest you do so before beginning Part 2.

Let's start working on our React Minesweeper game. First, rewrite index.js:

import React from 'react'
import ReactDOMClient from 'react-dom/client'
import App from './components/App.jsx'

const root = ReactDOMClient.createRoot(document.getElementById('root'))
root.render(<App/>)

And we'll move our <h1> into our App component: src/components/App.jx:

import React from 'react'

function App() {
  return <h1>Minesweeper</h1>
}

export default App
