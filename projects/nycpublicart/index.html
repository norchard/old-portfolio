NYC Public Art

Website for exploring NYC public art built in Javascript and Ruby

I made this website to visually explore temporary art exhibits around New York in a digital interface. I used D3 to plot the coordinates of temporary public art exhibits in NYC on a map, loading exhibit information and dynamically fetching an image for each. The static site is hosted on AWS S3. The original architecture for this project ran a Ruby script via a weekly cronjob to update the dataset on Monday mornings. However, this NYC Open Data dataset has not been updated since September 2018, so the site is no longer current. Visit publicartin.nyc to see the site or take a look at the code .
