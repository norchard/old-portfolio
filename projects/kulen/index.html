Kulen Outreach

Branding and website design

Really fun branding and web design project with the Division Of/ team for a non-profit running schools in Cambodia. The project won a Communicator Award in the Corporate Identity - Logo category. The logo was inspired by the iconic architecture of Ankor Wat. I was lucky enough to see the signage in person while traveling in Cambodia. You can learn more about Kulen and check out the website design at kulenoutreach.org .
